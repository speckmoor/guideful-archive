Was guideful technisch auszeichnete

guideful war ein spezialisierter Marktplatz für spirituelle Beratung. Über eine intuitive, kalendarische Buchungsoberfläche konnten Ratsuchende persönliche Gespräche mit qualifizierten Guides führen – via Chat, klassischem Telefon (Konferenzschaltung) oder browserbasiertem Video-Call (WebRTC).

Die Plattform wurde vollständig inhouse entwickelt – von der Produktlogik bis zur Zahlungsabwicklung. Alle Inhalte wurden semantisch verschlagwortet, um die passende Beratung schnell auffindbar zu machen. Die gesamte User Journey – von der Produktauswahl über das Gespräch bis zur Rechnungsstellung – war integriert.

Technologie-Stack

  • Frontend: Next.js 14 mit App Router (React-basiert)
  • Backend: Ktor (Kotlin)
  • Datenbank: MongoDB Atlas
  • Authentifizierung: Google Login via NextAuth (OAuth2)
  • CI/CD: GitHub Actions + Docker Dev-Setup, Deployment via Kubernetes (3 Pods)
  • Monitoring: Prometheus, Grafana, Uptime via HexTools

Besondere Features

  • Kalenderbasierte Buchungsoberfläche mit semantischer Produkt-Verschlagwortung
  • PDF-Rechnungen und Gutschriften (ZUGFeRD-kompatibel) vollständig automatisiert
  • Keine externen CMS – alle Inhalte und Adminfunktionen im Hauptsystem
  • KI-gestützte Produktentwicklung mit Claude, Gemini CLI und Browser-CoPilots
  • Automatisiertes End-to-End Testing mit dem eigenen Micro-SaaS TestCortex
  • KI-gestützte SEO-Content-Erstellung & Outreach-Automatisierung
